# 2RU-2RO

Developed by [alpmobs](https://alpmobs.webflow.io/)

## Overview

2RU-2RO is an image pin annotator that lets users upload an image, place pins, add notes, and export the annotated result as a PNG.

## Features

- Upload an image for annotation
- Add unlimited pins by clicking on the image
- Reuse title, description, and pin color for new pins
- Edit pin content directly from the image popover
- Drag pins to reposition them
- Show pin previews with title, description, and pin number
- Manage pins from the sidebar list
- Delete individual pins or clear all pins
- Customize pin color with matching popover border color
- Download the annotated image as a PNG
- Prevent download when no pin content has been added
- Keep the export styling aligned with the on-screen popup preview
